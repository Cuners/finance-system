# Финансовая система

Современное веб-приложение для управления личными финансами, позволяющее отслеживать бюджет, транзакции и категории расходов.

## Используемые технологии

### Backend
- **.NET 10.0** - основная платформа разработки
- **ASP.NET Core** - веб-фреймворк
- **Entity Framework Core** - ORM для работы с базой данных
- **JWT (JSON Web Tokens)** - аутентификация и авторизация
- **Redis** - кэширование данных
- **SQL Server** - реляционная база данных
- **Swagger/OpenAPI** - документация API
- **MassTransit** - библиотека для работы с шиной сообщений (для интеграции с RabbitMQ)
- **SignalR** - библиотека для работы с WebSocket и реальным временем


### Frontend
- **React 19** - библиотека для построения пользовательского интерфейса
- **TypeScript** - строгая типизация
- **Vite** - сборщик проекта
- **WebSocket API** - для взаимодействия с сервером в реальном времени

### Инфраструктура
- **Docker** - контейнеризация приложения
- **Docker Compose** - оркестрация контейнеров
- **Nginx** - веб-сервер и обратный прокси
- **Redis** - сервер кэширования
- **RabbitMQ** - шина сообщений для асинхронной обработки задач и уведомлений (через MassTransit)
- **WebSocket** - технология для двунаправленной связи в реальном времени (для уведомлений)

## Как запустить проект

### Предварительные требования
- Установленный Docker и Docker Compose
- Node.js (для локальной разработки фронтенда)

### Запуск через Docker Compose (рекомендуется)

1. Клонируйте репозиторий:
```bash
git clone https://github.com/Cuners/finance-system.git
cd finance-system
```

2. Запустите все сервисы:
```bash
docker-compose up --build
```

#### Запуск frontend части

1. Перейдите в папку client:
```bash
cd client
```

2. Установите зависимости:
```bash
npm install
```

3. Запустите приложение:
```bash
npm run dev
```
Веб-приложение будет доступно по адресу http://localhost:5173

## Основные функции

- Регистрация и аутентификация пользователей
- Создание и управление бюджетами
- Отслеживание транзакций
- Категоризация расходов
- Просмотр статистики и аналитики
- Система уведомлений (реализованная с использованием RabbitMQ и WebSocket)
- Создание документации
